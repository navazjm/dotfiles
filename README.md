# MacOS Dotfiles

Welcome to my macOS dotfiles branch! This setup is tailored for a productive and minimal macOS development environment.

## ⚙️ Tools & Configurations

| Tool             | Description                                         | Config File                                             |
|------------------|-----------------------------------------------------|---------------------------------------------------------|
|  Window Manager  | [Yabai](https://github.com/koekeishiya/yabai)       | [`.yabairc`](./.yabairc)                                |
|  Hotkey Daemon   | [skhd](https://github.com/koekeishiya/skhd)         | [`.skhdrc`](./.skhdrc)                                  |
|  Terminal        | [Alacritty](https://github.com/alacritty/alacritty) | [`alacritty.toml`](./alacritty/alacritty.toml)          |
|  Text Editor     | [Neovim](https://github.com/neovim/neovim)          | [`init.lua`](./nvim/init.lua)                           |
|  Shell           | [Zsh](https://github.com/zsh-users/zsh)             | [`.zshrc`](./.zshrc)                                    |
|  Multiplexer     | [Tmux](https://github.com/tmux/tmux)                | [`.tmux.conf`](./.tmux.conf)                            |
